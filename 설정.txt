프로그램 시작후

1. 프롬포트 카테고리 수정 또는 추가.
2. 카테고리 안에 프롬포트 수정 또는 추가.
3. 확인용 전체 카테고리와 프롬포트 같이 출력.
4. 입력용 전체 프롬포트만 출력.
5. 프로그램 종료.

출력후 선택 대기. 
선택은 앞의 번호로 받음.

1. 프롬포트 카테고리 수정 도는 추가 선택시
현제까지 생성된 카테고리 출력.
생성되있는 카테고리 입력시 카테고리 이름 수정.	
d를 입력하면 삭제.
생성되있지 않은 카테고리 입력시 카테고리 추가.
공백이나 스페이스 엔터등 글자가 없고 빈칸으로 보이는 입력들은 입력시 잘못된 입력입니다로 예외처리.
b를 입력받으면 뒤로가기.
뒤로가기를 하기전까지는 계속해서 프롬포트 카테고리 수정 도는 추가가 돌아감.

2. 카테고리 안에 프롬포트 수정 또는 추가 선택시 
현제 만들어진 카테고리를 하나씩 전부 출력한후 카테고리 안에 프롬포트 수정 또는 추가 추가할 카테고리 입력 받기.
공백이나 스페이스 엔터등 글자가 없고 빈칸으로 보이는 입력들은 입력시 잘못된 입력입니다로 예외처리.
입력 받은후 그 카테고리 안의 항목들 전부 출력. (출력 방식은 카테고리1, 카테고리2, 카테고리3 의 방식)
생성되있는 프롬포트 입력시 프롬포트 수정.
생성되있지 않은 프롬포트 입력시 프롬포트 추가.
공백이나 스페이스 엔터등 글자가 없고 빈칸으로 보이는 입력들은 입력시 잘못된 입력입니다로 예외처리.
생성되 있는 프롬포트 입력후 d를 입력하면 프롬포트 삭제.
b를 입력받으면 뒤로가기.
뒤로가기를 하기전까지는 계속해서 카테고리 안에 프롬포트 수정 또는 추가가 돌아감.
뒤로가기를 했을 경우 전체 카테고리를 하나씩 전부 다시 출력하여 카테고리를 확인하게 함.

3. 확인용 전체 카테고리와 프롬포트 같이 출력 선택시
카테고리 : 프롬포트, 프롬포트, 프롬포트, 프롬포트
그다음 카테고리 : 프롬포트, 프롬포트, 프롬포트, 프롬포트
와 같이 출력.
b를 입력받으면 뒤로가기.

4. 입력용 전체 프롬포트만 출력 선택시 입력되있는 모든 프롬포트를 처음부터 카테고리를 빼고
프롬포트, 프롬포트, 프롬포트, 프롬포트
와 같이 출력.
b를 입력받으면 뒤로가기.

5. 프로그램 종료 선택시
정말로 프로그램을 종료하시겠습니까?
출력후 확인 또는 뒤로가기 선택 대기.
확인 입력시 프로그램 종료.
b를 입력받으면 뒤로가기 선택시 뒤로가기.
